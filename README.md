# TetrisClone

A Tetris clone made with C++ and SFML library

## Controls
- **A/D**: move block left/right
- **S**: &nbsp;&nbsp;make block fall faster
- **W**: &nbsp;&nbsp;rotate block
- **P**: &nbsp;&nbsp;pause game
- **Esc**: quit game
